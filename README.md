# Electricity Consumption Analysis

 Project Title
**Plugging into the future - Electricity Consumption Analysis**

 About The Project
This project analyzes electricity consumption data from `Consumption.csv` to help users understand and reduce their electricity bills. 

We created an interactive dashboard to visualize daily, weekly, monthly usage and device-wise consumption.

 Objectives
- Analyze electricity usage patterns
- Visualize data using interactive charts
- Predict monthly bill amount
- Provide energy saving tips to users

## 🛠️ Technologies Used

- **Data Source**: Consumption.csv
- **Platform**: GitHub
